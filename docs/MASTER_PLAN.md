## 2. Technical Architecture (Shopify App + WP Plugin)

### Overview
- **Frontend:** React / Next.js
- **Backend:** Node.js (API), PostgreSQL
- **AI Layer:** OpenAI + local rules engine
- **Shopify Integration:** App Bridge, Polaris UI, Admin API, Webhooks
- **WordPress Integration:** Plugin with REST API bridge to your backend

### Architecture Breakdown
1. **Browser/Plugin → API Gateway**
2. **Auth Layer → User Mapping**
3. **Data Fetcher → Shopify/WP Content Sync**
4. **AI Processor → Summary/FAQ/Rewrite Engine**
5. **Scoring Engine → LLM Visibility Score™**
6. **File Generator → llms.txt + llms.json**
7. **Storage → Postgres (content snapshots + scores)**
